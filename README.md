## 3-Step Deployment (Takes 5 Minutes)

### Step 1: Create a GitHub Repository
1. Go to https://github.com/new
2. Name: `anglotec-tools` (or whatever you prefer)
3. Make it **Public**
4. Click **Create repository**

### Step 2: Upload These Files
1. Click **"uploading an existing file"** link
2. Drag and drop ALL files from this folder (including subdirectories)
3. Click **Commit changes**

### Step 3: Enable GitHub Pages + Your Domain
1. Go to **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **GitHub Actions**
3. Go to **Settings** → **Pages** again
4. Under **Custom domain**, enter: `tools.anglotec-ai.com`
5. Click **Save**
6. Check **Enforce HTTPS**

### Step 4: Add DNS Record (in IONOS)
1. Log into your IONOS account
2. Go to **Domains** → **anglotec-ai.com** → **DNS**
3. Add a **CNAME** record:
   - Host: `tools`
   - Points to: `YOUR_GITHUB_USERNAME.github.io` (replace with your actual username)
   - TTL: Default
4. Save and wait 5-15 minutes

Done! Your site will be live at `https://tools.anglotec-ai.com`
