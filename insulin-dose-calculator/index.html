📖 Insulin Calculation Formulas

500 Rule ( Estimate Carb Ratio)

  • Carb Ratio = 500 ÷ Total Daily Insulin
  • Example: TDI 50 units → 500/50 = 1:10 ratio

1800 Rule (Estimate Correction Factor)

  • ISF = 1800 ÷ Total Daily Insulin
  • Example: TDI 50 units → 1800/50 = 36 mg/dL/unit

Total Daily Insulin (TDI)

  • Type 1 DM: 0.5-0.6 U/kg/day
  • Type 2 DM: 0.3-1.0 U/kg/day (varies widely)
  • Basal: ~50% of TDI
  • Bolus: ~50% of TDI (divided among meals)

Target Blood Glucose Ranges

  • Pre-meal: 80-130 mg/dL
  • 2-hr post-meal: <180 mg/dL
  • Bedtime: 100-140 mg/dL
